# Radio 单选


* Radio

## Radio 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
disabled | bool | 是否被禁用 | 否 | false | - |
name | string | 表单项名 | 否 | '' | - |
checked | bool | 是否选中状态 | 否 | false | - |
type | string | 单选框样式类型 | 否 | `radio` | `radio` \| `button` |

## Radio 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 事件处理函数说明 |  
------- | ------- | ------- | ------- | ------- | ------- |
onClick | func | 点击事件 | 否 | null | 包含改属性的所有属性 |
