# Notice - 消息提示


* Notice

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
type | enum | 提示类型 | 是 | - | success 成功; warn 警告; error 错误 |
content | string | 提示语 | 否 | 空字符串 | - |
duration | number | 展示时长，单位为秒 | 否 | 2 | - |


## 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 事件处理函数说明 |  
------- | ------- | ------- | ------- | ------- | ------- |
onClose| func | duration 时间到后回调，主要用来让提示消失 | 否 | () => null | 无特殊参数 |
