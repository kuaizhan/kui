# Switch - 开关组件


*  Switch

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
disabled | boolean | 是否禁用 | 否 |  false | - |
name | string |  开关名称 | 否 | '' | - |
on | boolean | 开关是否开启 | 否 | false | - |

## 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 事件处理函数说明 |
onChange | func | 开关切换事件 | 否 | () => null | 事件回调函数第一个参数为包含on, name 属性的对象
