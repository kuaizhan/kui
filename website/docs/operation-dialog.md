# OperationDialog - 局部对话框组件


* OperationDialog

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |
------- | ------- | ------- | ------- | ------- | ------- |
hide | bool | 是否隐藏 | 否 | false | - |
title | string | 标题 | 否 | null | - |

------- | ------- | ------- | ------- | ------- | ------- |
onConfirm | func | 确定按钮点击事件 | 否 | null |   |
onCancel | func | 取消按钮点击事件 | 否 | null |   |
