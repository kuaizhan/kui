# DatePicker - 日期选择器组件


* DatePicker

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |
------- | ------- | ------- | ------- | ------- | ------- |
value | number or string | 时间戳或时间字符串 | 否 | 当前时间 | - |
