# Pager - 分页组件


* Pager

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
totalPage | number | 总页数 | 是 | - | - |
curPage | number | 当前页面，页面从 1 开始 | 是 | - | - |


## 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 事件处理函数说明 |  
------- | ------- | ------- | ------- | ------- | ------- |
onPageChange | func | 页码改变事件 | 否 | null | 事件第一个参数为页面值 |
