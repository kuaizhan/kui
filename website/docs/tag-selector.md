# TagSelector - 标签选择器组件


* TagSelector

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |
------- | ------- | ------- | ------- | ------- | ------- |
tags | array | tag列表 | 是 | [] | - |
default | number | 初始选中的标签 | 否 | 0 | - |
multi | bool | 多选还是单选 | false | 否 | - |

## 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 事件处理函数说明 |
------- | ------- | ------- | ------- | ------- | ------- |
onChange | func | 返回当前选中的tag | 否 | null | 回调函数参数为包含label, value属性的对象的数组 |
