# Confirm - 确认弹出框组件


* Confirm

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
hide | bool | 是否隐藏 | 否 | false | - |
confirmText | string | 确认按钮文案 | 否 | '确定' | － |
cancalText | string | 取消按钮文案 | 否 | '取消' | － |

## 方法

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
onConfirm | func | 确认按钮回调 | 否 | null | － |
onCancel | func | 取消按钮回调 | 否 | null | － |
