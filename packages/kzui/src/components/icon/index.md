# Icon - 图标


* Icon

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
iconClass | string | 图标class | 否 | '' | - |
type | string | 图标class，但不需传前缀kz-fe- | 否 | '' | - |
title | string | 鼠标 hover 上后的提示 | 否 | - | - |


## 事件
属性名 | 类型 | 描述 | 是否必须 | 默认值 | 事件处理函数说明 |  
------- | ------- | ------- | ------- | ------- | ------- |
onClick | func | 点击事件 | 否 | null | 无特殊参数 |
