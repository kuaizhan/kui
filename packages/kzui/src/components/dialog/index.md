# Dialog - 弹框组件


* Dialog

## 属性

属性名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
hide | bool | 是否隐藏 | 否 | false | - |
actions | array | 传入下方操作按钮 | 否 | [] | - |
title | string | 对话框标题 | 否 | '' | - |
portal | bool | 是否使用react16 portal | 否 | false | - |

## 方法

方法名 | 类型 | 描述 | 是否必须 | 默认值 | 字典 |  
------- | ------- | ------- | ------- | ------- | ------- |
onClose | func | 否 | 关闭回调 | () => {} | - |
